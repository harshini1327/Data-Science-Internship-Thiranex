# 🚀 Thiranex Data Science Internship

## 📖 About

This repository contains all projects and assignments completed during my **Data Science Internship at Thiranex**.

The internship focuses on developing practical skills in:

* Data Cleaning and Preprocessing
* Data Visualization
* Exploratory Data Analysis (EDA)
* Machine Learning and Predictive Modeling
* Real-World Data Science Applications

---

## 📋 Internship Tasks

### 📊 Task 1: Data Cleaning & Visualization Project

**Objectives:**

* Handle missing values, duplicates, and outliers
* Perform data preprocessing using Pandas
* Create visualizations using Matplotlib and Seaborn
* Generate meaningful insights from the dataset

**Skills Gained:**

* Data Cleaning
* Data Wrangling
* Data Visualization
* Data Storytelling

---

### 🤖 Task 2: Predictive Modeling Using Machine Learning

**Objectives:**

* Build and train machine learning models
* Evaluate model performance
* Apply algorithms such as:

  * Linear Regression
  * Decision Trees
  * Random Forest
* Visualize model results and performance metrics

**Skills Gained:**

* Supervised Learning
* Model Training
* Model Evaluation
* Performance Analysis

---

### 📈 Task 3: Exploratory Data Analysis (EDA)

**Objectives:**

* Perform statistical analysis
* Identify trends and patterns
* Analyze correlations between variables
* Present findings through visual reports

**Skills Gained:**

* Data Exploration
* Statistical Analysis
* Insight Generation
* Reporting

---

### 🌍 Task 4: Real-World Data Science Project

**Objectives:**

* Work with domain-specific datasets
* Apply end-to-end data science workflows
* Generate actionable insights and predictions
* Present conclusions through reports and dashboards

**Skills Gained:**

* Problem Solving
* Domain Analysis
* End-to-End Project Development
* Business Insights
